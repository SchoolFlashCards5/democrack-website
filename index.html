About

This project is an educational demonstration of how brute force and dictionary approaches work; only for learning purposes, not for real password cracking.

What’s Included

  • democrack.py — main script
  • democrack-mildlyfaster.py — optimized variant
  • democrack-withcomments.py — more explained version
  • democrack-gui.py — GUI version
  • commons.txt — common password list
  • icon.png — project icon for GUI version
